一. 获取路径名的目录部分
dirname NAME
dirname: strip last component from file name. Output NAME with its last
non-slash component and trailing slashes removed; if NAME contains no /'s,
output '.' (meaning the current directory).  例如:
dirname app/Phone.apk 会输出"app". dirname Phone.apk 会输出"."

二. 获取文件的后缀名
使用字符串的"最大匹配"特性来获取文件的后缀名,例如 ${filename##*.} 将会在
filename中一直查找'.'这个字符,直到最后一次匹配为止,最后匹配的就是后缀名.
例如: name="core.cpp"; echo ${name##*.} 会输出cpp.
当${filename##*}返回的值等于$filename的值时,表示该filename不带有后缀名.如:
name="pppd"; echo ${name##*.} 会输出pppd.

三. 获取路径名的文件名部分
basename NAME [SUFFIX]
basename: strip directory and suffix from filenames. Print NAME with any
leading directory components removed. If specified, also remove a trailing
SUFFIX. man basename手册中提供了两个例子,如下:
basename /usr/bin/sort 会输出sort. basename include/stdio.h .h会输出stdio
